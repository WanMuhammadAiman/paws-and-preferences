## What it does

- Shows a stack of random cat images from Cataas.
- User interactions:
  - Swipe right ‚Üí like  
  - Swipe left ‚Üí skip  
  - Buttons available for both desktop + mobile  
  - Arrow keys work on desktop (left/right)
- After all cats are viewed:
  - Shows total liked
  - Displays the liked cats in a simple grid
  - Option to restart and get a new batch
